# 分類器の学習用ソースコード

## 構成
- naivebayes_bernoulli.py: ベルヌーイモデルを用いたナイーブベイズ分類器
- naivebayes_polynomial.py: 多項モデルを用いたナイーブベイズ分類器
- scraping_URL: 記事一覧ページから各記事のURLを抽出するプログラム
- scraping_text: 記事から本文を抽出するプログラム

## 分類器
ベルヌーイモデルと多項モデルを実装した
文章のうちmecabによる形態素解析から名詞のみを抽出し、素性とした
